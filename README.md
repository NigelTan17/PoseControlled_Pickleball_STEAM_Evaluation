# Pose-Controlled Pickleball (Unity + Python)

A webcam-based pose-controlled pickleball prototype built with Unity and Python (MediaPipe).  
This repository also includes the analysis workflow used to evaluate user perception of STEAM-related value and engagement.

## Repository structure
- `Unity_Game/` - Unity project (Assets, Packages, ProjectSettings)
- `Python_Pose/` - Pose tracking + UDP streaming scripts
- `Python_Analysis/` - Survey analysis notebooks (domain scoring, reliability, group comparisons, correlation)
- `Dataset/` - Collected datasets from participants

## Environment
- Unity: 6000.0.33f1
- Python: 3.10
- Libraries: MediaPipe, OpenCV, NumPy, Pandas, SciPy
